== Description ==

Oracle Brains Social Book Marker adds user-submission text links at the bottom of each post for various social bookmarking sites which allow your visitors to easily submit them in a number of social bookmarking sites.

Supporting sites:
* Del.icio.us
* Reddit
* Slashdot it
* Digg
* Facebook
* Technorati
* Google
* Stumble
* Window Live
* Tailrank
* Furl
* Netscape
* Yahoo
* Blinklist



== Installation ==

1. Upload 'ob-textonly-social-bookmarker.php' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Upgradation ==
1. Simple follow step one of Installation and overwrite the existing file.

Thats it!
